# 函数

## 函数

- **定义**

  def关键字

  return  返回

- **调用**

  function()

- **return**

  返回结果

## 函数的参数

- **必备参数**

  `def func(a,b)`

- **默认参数**

  `def func(a=1,b=2)`

- **不定长参数**

  - 位置参数    *args

  - 关键字参数  **kwargs    键值对

  - 解包

    `*args,a,b,d = {1,2,3,4,5,6,7,8}`

## 内置函数

- enumerate()
- eval()
- filter()
- map()
- zip()
